# 🛡️ AmneziaWG Install for Wiren Board

Скрипт для быстрой и полностью автоматической установки туннеля **AmneziaWG** на контроллерах умного дома Wiren Board.

**AmneziaWG** — это форк WireGuard с обфускацией (защитой от систем глубокого анализа трафика DPI). В отличие от классических L2TP/IPsec или обычного WireGuard, трафик AmneziaWG маскируется под обычный UDP-шум. Это идеальное решение для коммерческих инсталляций и щитов, работающих через LTE-модемы или провайдеров, которые агрессивно блокируют VPN-соединения.

## 🌟 Особенности скрипта

- **Установка в одну команду:** Никакой ручной возни с созданием директорий, выдачей прав `chmod` или правкой системных файлов.
- **Автоматическая компиляция:** Скрипт сам скачивает актуальный компилятор Go от Google (игнорируя устаревшие пакеты Debian 11), выкачивает исходники с GitHub и собирает ядро под архитектуру контроллера.
- **Умная защита конфигурации (Smart Patching):** - Скрипт защищает контроллер от потери интернета. Даже если вы вставите конфиг с маршрутизацией `AllowedIPs = 0.0.0.0/0`, скрипт сам вычислит подсеть туннеля и подменит маршрут (например, на `10.8.0.0/24`), чтобы в VPN шел только трафик управления, а не весь интернет щита.
  - Автоматически прописывает `PersistentKeepalive = 25`, чтобы провайдеры не "убивали" неактивные туннели.
- **Идемпотентность:** Если скрипт запускается повторно, он пропускает этап долгой компиляции и просто мгновенно обновляет конфигурацию и перезапускает интерфейс.

## 🚀 Установка и использование

1. Подключитесь к вашему контроллеру Wiren Board по SSH.
2. Скопируйте и выполните следующую команду:

```bash
bash <(curl -sL [https://raw.githubusercontent.com/Batyushin/wb-awg-install/main/install_awg.sh](https://raw.githubusercontent.com/Batyushin/wb-awg-install/main/install_awg.sh))
